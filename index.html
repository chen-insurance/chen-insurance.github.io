חיבור למסד הנתונים

הזיני את פרטי ה-Supabase שלך. הפרטים נשמרים רק בדפדפן שלך ולא נשלחים לשום מקום אחר.

1. כנסי ל־supabase.com → פרויקט שלך
2. לכי ל־ Settings → API
3. העתיקי את Project URL ואת anon public key

זה המפתח הציבורי בלבד — בטוח לשמור בדפדפן
לא הצלחנו להתחבר. בדקי שה-URL והמפתח נכונים.
